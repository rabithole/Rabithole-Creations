Vista Park Campground

Vista Park is an RV, tent and yurt campground located in the quaint little town of Skamokawa Washington. Operated by Port #2 of Wahkiakum County this project was a complete redesign. With great views and an amazing beach, I saw the need for extensive photography. Photographs were initially taken in RAW format then processed in Photoshop. Photoshop was used to process photos, create the Vista Park logo, park map, panoramas in the events page and the 360 panorama on the park photos page.

I created a park map using Photoshop to create an overlay over a Google Earth image. The maps layout is geographically accurate do to the method used in its creation. The printable map used by the office was given a few adjustments to aid the park staff in giving directions to visitors.

Much time was spent on the navigation as it is entirely done with CSS. I utilized code I had used in a class project. It was an extensive learning experience with nested lists and hidden elements. The right color combination was key as it needed to work on every page of the site.

I spent my summer 2014 term intensively applying SEO techniques to this site. Utilizing Google Webmaster tools and eventually integrating some Micro data using schema.org. I acquired tons of data and was able to increase traffic to the site with a combination of Facebook marketing and an effective landing page. The data showed site visitors predominately landed on the main page first and progressed to key parts of the site to find various camping information they might need for their stay. The scope of this project didn't allow me to explore other social media outlets. Something I know is important, depending on the project.

County Line Park

County Line Park is also owned by Port #2 of Wahkiakum County. A much smaller park that is favored by fisherman due to it's close access to the Columbia River. It did not require a site as extensive as Vista Park or a separate business image. I created a sub domain for this site, both to save money and as a learning exercise I utilized the templates I created for Vista Park to quickly create this site. By doing so I kept a common theme for Port #2 in an effort that visitors might make the connection that these two parks are part of the same operation.
